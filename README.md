Параметры командной строки:

Perceptron.exe <полный путь до папки с тренировочными и тестовыми данными> <Число нейронов в скрытом слое(необязательно, 300 если не указывать)> <Скорость обучения(необязательно, 0.1 если не указывать)>

Примеры запуска:

Perceptron.exe C:\Documents\dataset 

Perceptron.exe C:\Documents\dataset 400

Perceptron.exe C:\Documents\dataset 300 0.2
